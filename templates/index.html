Features

Complete patient management package powered by ML and Robotics

Medicine Dispensing Robot

Medifi is powered by a robot called Medibot that helps the nurse to dispense medicine to the patients.

Patient management Software

An online cloud based Patient Management software is used to store and collect the data from the patients.

Heartbeat Variation Detection

Medibot has an inbuilt stethescope that can be used to detect heartbeat anomalies using machine learning

X-Ray anomaly detection and diagnosis

When a user uploads a chest x-ray our ML Algorithms detect the presence of known anomalies to assist diagnosis

Rfid based authentication

Every patient is awarded an RFID tag that acts as a key for all his data. It simplifies authentication and data access across multiple hospitals

Data management functionality

External agents like pharmacies can access the patient data history only with the the permission form the Doctor
